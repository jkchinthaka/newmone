# MaintainPro Mobile

This folder hosts the Flutter application for iOS and Android clients.

## Run locally

1. Install Flutter SDK (stable channel) and verify with `flutter doctor`.
2. From this directory run `flutter pub get`.
3. Start the app with `flutter run`.

## Platform scaffolding

- `android/` is reserved for Android-native project files.
- `ios/` is reserved for iOS-native project files.

If you need to regenerate native folders, run `flutter create .` from `apps/mobile`.
